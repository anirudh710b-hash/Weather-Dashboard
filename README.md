## 📖 About the Project

Weather Dashboard is a simple and interactive web application that allows users to search for any city and instantly view live weather information.

The application fetches real-time weather data from a Weather API and displays essential weather details in a clean and user-friendly interface.

---

## ✨ Features

- 🌍 Search weather by city name
- 🌡️ Live temperature updates
- 💧 Humidity information
- 💨 Wind speed
- ☁️ Weather condition
- 🌅 Dynamic weather icons
- ⚡ Fast API response
- 📱 Fully Responsive Design
- 🎨 Clean User Interface
- ❌ Invalid city error handling

## 🚀 Installation

### Clone the Repository

```bash
git clone https://github.com/anirudh710b-hash/Weather-Dashboard.git
```

### Open the Project

```bash
cd Weather-Dashboard
```

Now simply open

```text
index.html
```

in your browser.

---

## 🔑 API Configuration

Replace your API key inside **script.js**

```javascript
const API_KEY = "YOUR_API_KEY";
```

with your own Weather API key.
